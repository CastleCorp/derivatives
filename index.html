DERIVATIVES

Derivatives

Constant

f(x) = c (ex: f(x) = 2)

f′(x) = 0

Linear

f(x) = mx

(ex: f(x) = 2x)

f′(x) = 2

Power

f(x) = x^n

(ex: f(x) = x^2)

f′(x) = 2x^(2-1)

Exponential

f(x) = a^x

(ex: f(x) = 2^x)

f′(x) = ln(a) * a^x

Log

f(x) = ln(x)

f′(x) = 1/x

Trig (sin)

f(x) = sin(x)

f′(x) = cos(x)

Trig (cos)

f(x) = cos(x)

f′(x) = -sin(x)

Trig (tan)

f(x) = tan(x)

f′(x) = sec(2^x)

Properties

f(x) + g(x)     →     f′(x) ± g′(x)

f(x) * g(x)     →     (f′(x) * g(x)) + (g′(x) * f(x))

f(x) ÷ g(x)     →     ( (f′(x) * g(x)) - (g′(x) * f(x)) ) ÷ [g(x)]^2

f[g(x)]     →     f′[g(x)] * g′(x)

cf(x)     →     cf′(x)
